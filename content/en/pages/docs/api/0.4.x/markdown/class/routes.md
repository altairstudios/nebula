#### .routes( app )
> *@param* **app** _{Express()}_  - Express app instance  
> _@return_ **this** 

Adds bindings for the keystone routes

```javascript
var app = express();
app.use(...); // middlewareroutes, routes, etc. should come before keystone is mounted / started
keystone.routes(app);
```
