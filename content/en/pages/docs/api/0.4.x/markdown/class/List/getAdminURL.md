#### List.getAdminURL( item )
> *@param* **item** _{String}_ - item  
> _@return_ **String**    

Gets the Admin URL to view the list (or an item if provided)  

```javascript
	var listURL = list.getAdminURL()
	var itemURL = list.getAdminURL(item)  
```
